Перед запуском надо установить:
pip install requests

Открываем файл config.json
- в "coins" вводите названия монет которые нужно чекать на связки сразу после запуска скрипта
- в "exchanges-filter" вводите true (чекать только биржи прописанные в "exchanges") или false (чекать все биржи которые есть на coinmarketcup)
- в "exchanges" прописываете названия бирж которые нужно чекать (работает только когда "exchanges-filter" = true)
Остальное можете оставить как есть.

Запускаете скрипт
Скрипт автоматически чекает на связки монеты из конфига, далее можете вводить названия других монет и чекать их связки
